## Overview
The CPU is the "brain" of the computer, responsible for executing instructions and performing calculations.

## Key Concepts
- **Clock Speed:** Measured in GHz; determines how many cycles per second the CPU can perform.
- **Cores & Threads:** More cores = better multitasking. Threads represent virtual cores.
- **Cache Levels:**
  - L1: Fastest, smallest
  - L2: Medium
  - L3: Largest, shared across cores
- **Architecture:** x86, x64, ARM
